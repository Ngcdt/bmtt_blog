JavaScript: ES6+ và cú pháp hiện đại

Trần Ngọc Đạt

December 30, 2025

ES6+ thay đổi cách viết JavaScript, giúp mã ngắn gọn và dễ đọc hơn. Nội dung:

  • let / const vs var
  • Arrow functions, template literals, default params
  • Destructuring, rest/spread, và modules (import/export)
  • Promise, async/await (liên quan tới phần bất đồng bộ)

Kèm ví dụ so sánh giữa cách cũ và cú pháp hiện đại để bạn luyện tập.
