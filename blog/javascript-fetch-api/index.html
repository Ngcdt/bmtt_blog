JavaScript: Fetch API và gọi REST

Trần Ngọc Đạt

December 30, 2025

Fetch API là phương thức hiện đại để gọi HTTP từ trình duyệt. Bài viết bao gồm:

  • Sử dụng fetch để gọi endpoint GET/POST
  • Xử lý JSON, bắt lỗi mạng và timeout cơ bản
  • Cách cấu hình headers (Authorization, Content-Type)
  • Thực hành: gọi REST API và hiển thị dữ liệu lên DOM

Mẹo tối ưu: dùng AbortController để huỷ request khi cần.
